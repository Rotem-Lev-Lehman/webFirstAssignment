Question 1 - What are 3 main differences between web 1.0, web 2.0 and web 3.0 by your opinion?

To answer this question, I will first explain a little bit what is web 1.0, web 2.0 and web 3.0.

Web 1.0 - It contains mostly static pages and web sites, their data updates only once in a while...
The search in web 1.0 was pretty basic, and not very accurate.
There was a basic design to the pages, and they were not interactive.

Web 2.0 - There are now social networking sites, like Facebook.
There are blogs and micro-blogs, like Twitter.
There are sites that allow users to contribute content, like Wikipedia.
There are also sites that let users share content, like YouTube!
There is also an improvement in the search, now there is also a semantic search, so there are better results :)
