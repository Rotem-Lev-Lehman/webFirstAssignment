Q&A


Question 1 - What are 3 main differences between web 1.0, web 2.0 and web 3.0 by your opinion?

To answer this question, I will first explain a little bit what is web 1.0, web 2.0 and web 3.0.

Web 1.0 - It contains mostly static pages and web sites, their data updates only once in a while...
The search in web 1.0 was pretty basic, and not very accurate.
There was a basic design to the pages, and they were not interactive.
Users passively receive information without being given the opportunity to post reviews, comments, and feedback.

Web 2.0 - There are now social networking sites, like Facebook.
There are blogs and micro-blogs, like Twitter.
There are sites that allow users to contribute content, unlike Web 1.0, Web 2.0 facilitates interaction between web users and sites, so it allows users to interact more freely with each other
There are also sites that let users share content, like YouTube!
There is also an improvement in the search, now there is also a semantic search, so there are better results :)

Web 3.0 - It is the "executable" phrase of Word Wide Web with dynamic applications, interactive services, and "machine-to-machine" interaction.
Its now all about the wisdom of the crowd!
Web 3.0 is about making the internet more indexed, with unified Semantics... its just like one big virtual Data-Base...
We use AI algorithms to discover trends on the web
We use Semantic search to get better results for our queries
We use IoT devices, cloud computing, etc...
Pretty awesome ha?

to conclude, lets put the main differences in a table:

subject \ web type Web 1.0 Web 2.0 Web 3.0
Interaction Not very interactive, users passively receive information. Users can contribute information, share content and feedback on content. There are now also "machine-to-machine" interactions, and user-to-machine interactions
Search Basic search, not very good results. There is also a semantic search, so there are better results, unnecessarily in the order\format that you wanted(as a user...) The results are better, and are personalized for each individual.
Users communication through the web No communication, each user can post his own site, but they don't realy communicate with each other, only upload their own data to their own sites... There are social networks, like Facebook. We also communicate with machines now, like IoT devices...

Question 2 - Give an example of a Use-Case that shows the differences between Web 1.0, Web 2.0 and Web 3.0.

I chose to investigate how would "checking what will be the weather tomorrow" be like in Web 1.0, Web 2.0 and Web 3.0:

In Web 1.0, the user will go to a weather website, and see the content that the site's manager has uploaded.

In Web 2.0, the user will search the querie "What would be the weather tommorow", and he will get as a result the weather wanted, with comments from other users about is this site reliable and should he belive this weather prediction.

In Web 3.0, the user will ask an IoT device, like Alexa, what will be the weather tommorow, and the device will tell the user the weather after checking with the most reliable site.
